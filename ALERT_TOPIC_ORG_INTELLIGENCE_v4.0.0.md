# ParleyMap v4.0.0 Influence Intelligence Expansion

The accepted map-first layout is preserved. This iteration adds sticky intelligence below the map: alerts, topic radar, recurring attendees, organization profiles, event-overlap matrix, organization penetration, topic migration, influence timeline and power geography.

Core durable model: Person <-> Event, with organization, role, year, speaker/attendee/organizer status, source URL, topic, theme, location and confidence.

Restricted or sensitive leads may guide search, but public map records still require safe public source packs.
