# Iteration Notes v0.2.0

## User-requested changes
- Provide reviewable HTML on every iteration.
- Provide one GitHub deployment zip with all project files and documents.
- Provide one recovery/handoff zip documenting discussion progress, development state, and targets.
- Use OSM mapping.
- Show movement of people.
- Support zooming, hover stops, and event information.
- Use a narrow Arial-compatible typography system.
- Use maximum three font sizes.
- Optimize layout for mobile and eventual app deployment.
- Keep cards inside the interface.
- Include proper legal and disclaimer language.

## Implemented
- Standalone `index.html` review prototype.
- Leaflet + OpenStreetMap map.
- Marker hover tooltips and click popups.
- Movement paths per person.
- Encounter rings for shared public events.
- Filter and sort controls.
- Selected stop card that stays inside the map panel on desktop and in page flow on mobile.
- Three-font-size design system.
- Built-in legal/editorial disclaimer.
- GitHub Pages workflow.
- Recovery package.

## Known limitations
- Data is synthetic.
- Leaflet and OSM tiles load from public CDNs/services.
- No backend, ingestion, authentication, editorial UI, or persistence yet.
- No real open-license images yet.
- No production legal review yet.
