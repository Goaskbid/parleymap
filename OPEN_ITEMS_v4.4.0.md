# ParleyMap v4.4.0 recovery summary

Current anchor remains unchanged: top-10 map default, map first, selected profile on the right, broader intelligence below the map.

This release is a finishing/presentation pass. It does not change the core map or roadmovie functionality. It improves readability and interaction in the below-map intelligence layer:

- institutional networks show named faces and clickable representatives;
- signals include event names, date/city, participant faces, source document pills and news search pills;
- event catalog rows use short useful commentary instead of field tags;
- organization profiles and power geography show names next to faces;
- overlap matrix adds cluster cards and a compact scrollable matrix;
- organization penetration index is now a readable lane/card layout;
- topic migration shows topic movement, dates, cities and rough intensity;
- influence timeline is a wide horizontal rail;
- relationship web can highlight selected leader edges and open contact details.

Keep the public-source boundary: no private locations, live tracking, hotels, leaked itineraries, aircraft-tracker-derived records or unsourced sightings.
