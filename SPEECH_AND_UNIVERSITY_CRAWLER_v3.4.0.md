# ParleyMap v3.4.0 Iteration Notes

## Anchor

v3.4 keeps the v3.2 layout as the fixed design baseline: map first, profile-only right rail, broader intelligence below the map, top-10 opening map, roadmovie controls and source-backed stop cards.

## Expansion

This version extends the v3.3 event pack with a dedicated speech-and-university layer. New records cover Federal Reserve, ECB, Bank of England, Bank of Japan, Bank of Canada and BIS public speeches. The build also adds more event records for Bilderberg, Jackson Hole, OECD, UN High-level Week, OPEC, IMF/World Bank, Davos, Paris AI Action Summit, Munich Security Conference and Milken Global Conference.

## Data additions

- Central-bank roadmovies now have more stop density: Stanford, Jackson Hole, Nashville, Dallas, New York, Arlington, Philadelphia, Sintra, Frankfurt, Geneva, Helsinki, London, Reykjavík, Tokyo, Osaka, Nagoya, Toronto, Vancouver, Calgary and Montréal.
- Added central bankers, G30 / BIS figures, energy-policy figures, OECD / UN / OPEC people and Bilderbergers into the mapped layer.
- Event agenda cards and institutional network rows remain below the map and are clickable / hoverable.

## Boundary

Only public speech pages, official event pages, public participant lists and host pages are used. The dataset is city-level and does not publish hotels, residences, leaked schedules, live movements or private details.
