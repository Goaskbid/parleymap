# ParleyMap v2.5.2 Iteration Notes

This iteration fixes the travel movie interaction. The public trail now reads as a roadmovie: public base to public stop, a return-home storyline where no official route is known, then the next public stop. Playback is slower by default and supports pause/play, restart, rewind, forward, latest, fast mode, and timeline scrubbing.

The opening map keeps the top-20 face map but adds a clickable Top 20 name list next to it, so users are not forced to hit tiny faces on the map. Hover cards are capped and compact. Map labels remain off by default to avoid overlapping text.

The 24-month crawler now scans monthly discovery windows rather than only the recent period. Official and host sources remain the publication path; discovery leads do not publish without verification.
