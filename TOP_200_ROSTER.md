# Summit Watch and Peace-Process Layer

## Purpose

The summit and peace-process layers turn the map into an influence-intelligence surface. They reveal where major public diplomacy is scheduled, where documented peace talks occurred, and where relationship edges are forming.

## Summit watch rules

A future summit can be displayed only when an official government, multilateral, host, corporation, royal household, or event organizer has publicly announced it.

Future precision defaults to city level. Venue precision is allowed only if the official source intentionally publicizes the venue and the safety review confirms that displaying it does not create inappropriate security risk.

## Peace-process rules

Peace-process records may include public summits, negotiations, ceasefire talks, humanitarian-corridor talks, prisoner-exchange talks, public bilateral peace meetings, and official statements tied to conflict resolution.

The interface highlights these records, but the language must stay sober. It should not claim outcomes that the sources do not support.

## Display behavior

- Peace-process markers receive a distinct marker/ring style.
- Peace-related event cards include a peace-process pill.
- Peace highlights are clickable ranking rows.
- Why This Matters summaries should explain historical context, participants, documented purpose, strategic relevance, and expected next public milestones.
