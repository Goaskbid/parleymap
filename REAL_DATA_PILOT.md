# Product Requirements Document: ParleyMap

## Vision

Create the best public-source map of where globally prominent public figures appear in official and public contexts, with concise explainers, verified source packs, and future public engagement watch cards.

## Primary users

- Journalists and editors.
- Policy analysts.
- Researchers.
- Corporate intelligence teams.
- Event planners and diplomatic watchers.
- Civically curious readers.

## Core jobs

1. Find where a person was last publicly documented.
2. See known future public engagements for the next week or 60 days.
3. Understand who appeared in the same public event cluster.
4. Compare travel intensity and meeting networks.
5. Read a short explainer with source links and confidence.
6. Audit the source and image licensing behind a card.

## MVP

- 250 public figures.
- 24-month rolling public appearance history.
- 60-day announced future engagement window.
- Map, list, search, filters, profiles, encounters, source packs, correction flow.
- Admin review for uncertain records.

## Long-term target

- 500-1000 public figures.
- Daily automation with source confidence.
- Annual roster review and promotion/office-change triggers.
- Public API and embeddable widgets after legal review.

## v2.0 product direction update

ParleyMap's primary value proposition is now influence intelligence, not location tracking. The map must always answer: why should the user care about this public movement or encounter?

Core V2 objects: Person, Organization, Location, Appearance, Encounter, RelationshipEdge, Summit, Signal, SourcePack, MediaAsset, MarketRelevance, ReviewState.

The product should prioritize source transparency, explainable rankings, institutional tone, legal/safety limits, and relationship intelligence over raw map density.
