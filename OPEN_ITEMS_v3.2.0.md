# Project recovery summary: ParleyMap v3.2.0

ParleyMap is an influence-intelligence interface for documented public appearances, meetings, summit windows and public call readouts. It is not a live tracker.

The v3.2.0 pass was a final audit pass aimed at deployment stability and usability. The opening map now defaults to a clean top-10 public-anchor view and offers top-20, all-top-200 and regional presets. Routes and call lines are hidden on the opening map; they appear only after a person or summit context is selected. The roadmovie start button now follows the selected figure and labels itself with the selected name. The right rail is reserved for the active profile, while most-travelled, connectedness, latest activity, future windows, peace talks and strategic-city rankings sit below the map.

The GitHub package contains the static site, data files, crawler scaffolding, nightly workflow, top-200 roster, source registries, legal/safety docs and recovery notes.
